# GitMan

Manager for local git repositories

## Installation
Clone the repo then use the
```shell
bin/install
```
command or execute the script content manually.
   
## Usage
```shell
symfony console user-admin
```
Create and administer user accounts.
```shell
bin/start
```
```shell
bin/stop
```
start and stop the environment.

## Testing

```shell
make tests
```
